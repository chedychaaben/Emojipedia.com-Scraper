Emoji Meaning

A symbol representing Blood Type B. Displayed as the letter B on a red background.

Sometimes used online to replace the letter C at the start of words in reference to the Bloods street gang's practice of turning words like cool into bool. This has expanded in some instances to starting any word with the letter B, or more specifically, this emoji which makes it more 🅱️lear that a letter has been replaced.

B Button (Blood Type) was approved as part of Unicode 6.0 in 2010 under the name “Negative Squared Latin Capital Letter B” and added to Emoji 1.0 in 2015.
