Emoji Meaning

A slightly open tabbed folder, as used to file papers in a cabinet or as an icon for an opened folder when using a computer. Most platforms depict a yellow folder, as manila, with the tab shown at the left.

Commonly used for various content concerning filing, organization, and work, especially at an office.

Apple’s folder is gray, Twitter’s is blue. Google’s design features multiple tabs, Microsoft’s a red tab.

Nearly identical in appearance and use to 📁 File Folder, which is closed.

Open File Folder was approved as part of Unicode 6.0 in 2010 and added to Emoji 1.0 in 2015.
